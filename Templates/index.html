Welcome to Book Recommendation System

This is a simple book recommender system that recommends books based on the similarity between the books. It uses a collaborative filtering approach to recommend books that are similar to the ones that the user has already read and rated.
In this system we have used K-means clustering algorithm to cluster books into different clusters based on similarity between the ratings of the books and used KNN algorithm with cosine similarity metric to recommend the book user searches.


Click the 'Recommend' link in the navigation bar to get started!
